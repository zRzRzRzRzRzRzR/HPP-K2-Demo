# HPP-BioHealth

## Install requirements

```bash
pip install -r requirements.txt
```

rename `.env_example` to  `.env` file with your OpenAI and K2-Think API keys.

## Run

```bash
python gradio_app.py
```

and it will start a gradio app at `http://0.0.0.0:7860`
